# Contributing Guidelines

Thank you for your interest in contributing to this repository!  
Please follow the steps below when you want to share any thought, idea, or suggestion.

## âœ¨ How to Contribute

### 1. **Fork this repository**
Click the **Fork** button on the top right and create your own copy of the repo.

### 2. **Create a new file**
Inside your fork:
- Create a new file under the `thoughts/` directory (if it doesn't exist, create it).
- Use a meaningful filename like:  
  `yourname-thought.txt` or `idea-title.md`

The file should contain the thought/idea you want to share about our friend ASHU or someone u cherish .

### 3. **Commit your changes**
Add your file and commit it with a message like:
`yourname-thought`
